# hanghaeMusic99 🎧
Chapter1 웹개발 미니 프로젝트 (항해뮤직99)

### music platform에서 가져온 실시간 Top 99위 까지의 노래에 자유롭게 코멘트를 작성하며 소통할 수 있는 웹입니다.

## 1. 제작기간 & 팀원소개
- 2022년 5월 9일 ~ 2022년 5월 12일
- 4인 1조 팀 프로젝트
  - 최지훈
  - 문희린
  - 송완준
  - 김형준

## 2. 사용 기술
`Back-end`
- Python 3
- Flask 2.1.1
- MongoDB

`Front-end`
- JQuery 3.5.1
- Bulma 0.9.2

`deploy`
- AWS EC2(Ubuntu 18.04 LTS)
